# DevArchive

Historical, no-longer-authoritative documents kept for provenance only. Nothing
here describes the current app or workflow. The live source of truth is at the
repo root: `CLAUDE.md` (workflow), `BACKLOG.md` (task queue), `FEATURES.md`,
`RESUME.md`, and the spec docs (`ARCHITECTURE.md`, `DATA_MODEL.md`, etc.).

- `CLAUDE_CODE_PROMPT.md` — the original "master prompt" used to bootstrap the
  build in an empty repo. The app is now built; the prompt references a `/docs`
  directory that no longer exists (docs live at the repo root).
- `HOW_TO_USE_WITH_CLAUDE_CODE.md` — one-time "create the repo from scratch and
  launch Claude Code" setup guide. Superseded by the `make progress` workflow in
  `CLAUDE.md`.
